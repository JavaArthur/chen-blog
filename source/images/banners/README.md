# 横幅图片文件夹

这个文件夹包含各个页面的横幅背景图片。

## 需要的文件：

### index-banner.jpg
- **用途**: 首页横幅背景
- **建议尺寸**: 1920×1080px
- **格式**: JPG
- **样式**: 高质量风景、抽象图案或品牌相关
- **占位图片**: https://picsum.photos/1920/1080?random=1

### default-banner.jpg
- **用途**: 默认横幅背景 (当页面没有设置特定横幅时使用)
