# 背景图片文件夹

这个文件夹包含网站的背景图片。

## 需要的文件：

### site-bg.jpg
- **用途**: 网站整体背景
- **建议尺寸**: 1920×1080px 或更大
- **格式**: JPG (文件大小控制在 500KB 以内)
- **样式**: 淡雅、低对比度，不影响文字阅读
- **占位图片**: https://picsum.photos/1920/1080?random=21&blur=2

### footer-bg.jpg
- **用途**: 页脚背景
- **建议尺寸**: 1920×400px
- **格式**: JPG
- **样式**: 与整体风格协调
- **占位图片**: https://picsum.photos/1920/400?random=22

## 设计建议：
- 使用模糊效果或低饱和度
- 避免过于鲜艳的颜色
- 确保与文字有足够的对比度
- 考虑使用渐变或纯色背景

## 性能优化：
- 背景图片文件大小控制在 500KB 以内
- 可以考虑使用 WebP 格式进一步压缩
- 为移动设备准备较小尺寸的版本
